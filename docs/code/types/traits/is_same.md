описание
--------
Мета-функция: сравнивает указанные типы.  
Возвращает true, если типы равны.  

```
std::cout<< tools::is_same<bool, bool>::value << '\n'; // 1
std::cout<< tools::is_same<long, bool>::value << '\n'; // 0
```
