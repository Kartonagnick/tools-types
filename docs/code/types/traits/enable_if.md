описание
--------
Мета-функция: реализует паттерн `SFINAE`  
Если первый аргумент - `false`, 
тогда шаблон не скомпилируется.  

```
template<class t>
tools::enable_if<(sizeof(t) > 1)> foo(T);
  ...
foo(100); // OK sizeof(int) > 1;
foo('a'); // ERROR sizeof(char) == 1
```
