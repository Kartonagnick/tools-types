описание
--------
Мета-функция возврвщает true, 
если указанный тип - лямбда.

Лямбда - функтор, у которого есть только один публичный `operator()`
Пример:  

```
const auto lambda = [](int){};

using result 
    = tools::is_lambda<decltype(lambda)>;

std::cout << result::value << '\n';
```
