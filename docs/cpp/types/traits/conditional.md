описание
--------
Мета-функция: выбор одного из двух типов по условию.  
Если условия true, тогда результат равен A, а иначе - B  

```
enum { expr = true };
using cond = tools::conditional<expr, A, B>;
using result = typename cond::type; // result is A
```
